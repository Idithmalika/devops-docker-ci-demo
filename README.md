# DevOps Docker CI/CD Demo 

![CI](https://img.shields.io/github/actions/workflow/status/Idithmalika/devops-docker-ci-demo/ci.yml?branch=main)

A simple Node.js application demonstrating **DevOps best practices** with:

- Docker & Docker Compose
- GitHub Actions CI/CD pipeline
- Automated testing and Docker image build



## Features

- Containerized Node.js app
- Automated CI pipeline on every push
- Easy to run locally with Docker
- Portfolio-ready DevOps project
